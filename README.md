# Multifunctional Arithmetic Unit (VLSI)

## Project Overview

This project presents the design, implementation, and verification of a 16-bit Multifunctional Arithmetic Unit using digital logic design and VLSI design methodologies.

The project was developed using Logisim for functional verification and Magic VLSI with IRSIM for layout generation, extraction, and simulation.

## Features

* 16-bit arithmetic operations
* Carry Propagate Adder (CPA)
* Carry Select Adder (CSA)
* Hierarchical digital design
* Functional verification
* Layout extraction and simulation

## Design Tools

* Logisim
* Magic VLSI
* IRSIM
